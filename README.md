# Multi-currency Bookkeeping App
This is a simple React Native app for tracking income and expenses in multiple currencies.

## Features
- Add income and expense records
- Choose currency: USD, CNY, EUR, JPY
- Dark mode support (auto switch based on system theme)
- View list of records

## How to Run

1. Clone the repo:
git clone https://github.com/mochinagi/bookkeeping-app.git

2. Install dependencies:
npm install

3. Run the app:
npx react-native run-ios
npx react-native run-android


## Tech Stack
- React Native
- TypeScript
- @react-native-picker/picker
